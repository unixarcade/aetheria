# Aetheria

**Multi-Modal Consciousness Loom**

A self-contained meditation, focus, and altered-state instrument that runs entirely in the browser. Aetheria weaves four modalities — binaural acoustics, paced breath (prana), generative visuals, and focus targeting — into a single loom you can tune in real time.

No accounts. No telemetry. No backend. Open the page and begin.

---

## Modalities

**Acoustic** — Binaural beat generator with selectable brainwave states (Delta, Theta, Alpha, Beta) and adjustable carrier frequency. Designed for headphones.

**Prana** — Configurable breath cadence (box, 4-7-8, custom) driving a synchronized visual breath orb.

**Vision** — Generative canvas visualizer responding to the active acoustic and breath state. Color, complexity, and velocity are tunable.

**Focus** — Stargate coordinate generator (random remote-viewing style targets) and optional session timer (5 / 15 / 30 minutes, or free).

---

## Use

1. Put on headphones (binaural effects require stereo separation).
2. Tap **Initialize** to start the audio context.
3. Tap the breath orb to begin a session.
4. Pull up the modality deck from the bottom to tune each layer.

Works on iOS Safari, Android Chrome, and desktop browsers. Designed mobile-first.

---

## Deployment

This is a single HTML file. Drop `index.html` at the root of a GitHub Pages repo and it's live. No build step, no dependencies, no configuration.

---

## Safety Note

Binaural entrainment and breath work can produce altered states. Don't use while driving or operating machinery. If you have a seizure disorder, consult a physician before using strobing or pulsing visuals.
